How to configure

  1. Go to https://console.developers.google.com/project
  2. Create a new project and call it whatever you want (ex: OctoberOAUTH)
  3. Once the project was created, go to:
    • API Manager
    • Credentials
    • Create credentials
    • OAuth client ID
  4. Configure "consent screen"(if necessary)
  5. On "Create client ID" screen, select "Web application"
  6. Fill:
    • Authorized JavaScript origins: (you can leave this field blank if you want)
    • Authorized redirect URIs: /creations/googleapi/google
  7. Click Create; a popup will appear with the "Here is your client ID" and "Here is your client secret", please, copy and paste in the fields below
    (now you can close Google Developer Console window)
  8. Save this form, and you're ready to login with your Google account.
